Qual a diferença entre linguagens de Programação Server Side e Client Side?

Existem dois tipos de linguagens desenvolvidas para internet:
Linguagens Client Side e as linguagens Server Side.
As linguagens Server Side são linguagens que o SERVIDOR entende.
Linguagens Client Side, que o navegador entende.
Essas linguagens são mais conhecidas como Back-end (Server Side)
e Front-end (Client Side).

O desenvolvimento Back-end cuida da parte interna da aplicação,
ou seja, uma estrutura lógica que o usuário não vê
mas que faz a aplicação funcionar.

Desenvolvimento Front-end cuida da parte externa da aplicação,
unindo design e tecnologia.
O Front-end desenvolve tudo o que o usuário vê e interage.
